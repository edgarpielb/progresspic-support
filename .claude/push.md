description: Commit all changes and push to GitHub
---

Please commit all changes and push them to GitHub. Follow these steps:

1. Check the current git status to see what files have changed
2. Stage all changes (modified, deleted, and new files)
3. Create a commit with a descriptive message that summarizes the changes
4. Push the commit to the remote repository

Make sure to:
- Review the changes before committing
- Write a clear, concise commit message
- Include the standard co-authored-by attribution
- Confirm the push was successful
