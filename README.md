# ProgressPic Support Site

This is the support website for the ProgressPic iOS app, hosted on GitHub Pages.

## Live URLs

- **Support Page:** https://edgarpielb.github.io/progresspic-support/
- **Privacy Policy:** https://edgarpielb.github.io/progresspic-support/privacy.html
- **Terms of Service:** [Apple's Standard EULA](https://www.apple.com/legal/internet-services/itunes/dev/stdeula/)

## Files

- `index.html` - Main support page with FAQ, features, and contact information
- `privacy.html` - Privacy policy detailing data practices
- `app-icon.png` - App icon for branding

## Deployment

The site is automatically deployed via GitHub Pages from the `main` branch.

### To Update

1. Make changes to the HTML files
2. Commit and push to the `main` branch:
   ```bash
   git add .
   git commit -m "Update support site"
   git push origin main
   ```
3. Changes will be live within a few minutes
